# 画像ファイル

このディレクトリに画像ファイルを配置してください:

- `profile.jpg` - プロフィール画像（About ページ用）
- `project1.jpg` - プロジェクト画像1
- `project2.jpg` - プロジェクト画像2
- `project3.jpg` - プロジェクト画像3
- `project4.jpg` - プロジェクト画像4
- `project5.jpg` - プロジェクト画像5
- `project6.jpg` - プロジェクト画像6

推奨サイズ:
- プロフィール画像: 500x500px（正方形）
- プロジェクト画像: 1200x630px（16:9 アスペクト比）

画像がない場合は、プレースホルダーとして灰色の背景が表示されます。
